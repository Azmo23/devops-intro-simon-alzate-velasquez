# devops-intro-simon-alzate-velasquez

La cultura DevOps (Desarrolo y operaciones) lo que busca en esta sociedad tan desarrollada que lo que busca cada vez mas las cosas rápidas, se busca hacer entregas continuas o progresivamente al usuario para entenerder sus capacidades y las necesidades que busca cubrir.

Una de sus claves es la automatizacion para realizar estos proyectos, porque lo que se busca con esto es que el humano intervenga menos en estos procesos tales como instalar entornos, empaquetar, tetear, etc

También trabajar bajo la filosofia DevOps nos facilita la recuperación en casos de emergencia ya que se va a tener un mejor control activo de los software y hardware y las tareas automatizadas tales como instalar, cambiar configuraciones o volver a una version anterios.

La cultura DevOps se basa en un ciclo productivo que comienza desde:

-Plan >>> Build >>> Continuous Integration >>> Deploy >>> Operate >>> Continuous Feedback

Despues del ultimo paso retoma a la planeación ya que es un ciclo repetitivo que mejora la calidad de la producción.

## Azure DevOps

Azure DevOps es una plataforma de Microsoft que proporciona un conjunto de servicios integrados para ayudar a los equipos de desarrollo de software a planificar, desarrollar, probar y entregar aplicaciones de manera más eficiente. Se basa en los principios de DevOps, que buscan unir a las personas, los procesos y la tecnología para ofrecer valor a los clientes de forma continua.

Entre sus principales características, Azure DevOps incluye:

- Azure Boards: Herramientas para la gestión de proyectos ágiles, tableros Kanban, backlogs y seguimiento de trabajo. Permite organizar tareas, errores, historias de usuario y más.

- Azure Repos: Repositorios de código fuente (Git y TFVC) para el control de versiones, colaboración en código y gestión de ramas.

- Azure Pipelines: Herramientas de integración continua (CI) y entrega continua (CD) que automatizan la compilación, prueba y despliegue de software en cualquier plataforma o nube.

- Azure Test Plans: Soluciones integrales para la gestión de pruebas, incluyendo pruebas manuales, exploratorias y automatizadas.

- Azure Artifacts: Herramienta para gestionar paquetes de software (Maven, npm, NuGet, Python) y compartirlos entre equipos.

En resumen, Azure DevOps es una suite completa que abarca todo el ciclo de vida del desarrollo de software, facilitando la colaboración, la automatización y la mejora continua.
